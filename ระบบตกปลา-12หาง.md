# ระบบตกปลา — คอมมู 12 หาง (Design Spec)

> เอกสารสรุปแนวคิดระบบตกปลา สำหรับนำไปพัฒนาต่อ
> **หลักการสูงสุด: ระบบตกปลาต้องไม่กระทบเศรษฐกิจคอสตูม/Jil ที่เป็นรายได้จริงของเกม**

---

## 1. เป้าหมายจริงของระบบ (อ่านก่อนออกแบบอะไรทั้งนั้น)

หน้าที่ของตกปลา **ไม่ใช่** การทำเงินหรือเป็นเครื่องปั๊มรางวัล

หน้าที่จริงคือ **เครื่องรั้งคนให้อยู่ใน world chat** — ให้คนมีอะไรทำระหว่างพิมพ์คุย ทำให้ห้องไม่เงียบ สร้างจังหวะและเรื่องคุย ("เพิ่งได้ตัวหายาก", "ใครมีเหยื่อบ้าง", "เหลือ 50% แล้วปลาหลุด!")

**ตัวชี้วัดความสำเร็จ:** คนอยู่นานขึ้น / คุยกันเยอะขึ้น
**ไม่ใช่:** ระบบตกปลาทำเงินได้เท่าไร (เงินยังมาจากขาย Jil เหมือนเดิม)

> สรุปหน้าที่: **ขาย Jil = ทำเงิน / ตกปลา = ทำให้มีคนอยู่ให้ทำเงิน**

---

## 2. กฎเหล็กเรื่องสกุลเงิน (เส้นเป็นเส้นตาย)

### ❌ ตกปลาห้ามผลิต Jil เด็ดขาด

Jil = สกุลเงินเติมเงิน ใช้ซื้อคอสตูมพรีเมียม (สี/หน้า/อาวุธ/หมวก) — **เป็นรายได้จริงของเกม**

ถ้าตกปลาปั๊ม Jil ได้ → คนไม่ต้องเติมเงิน ตกปลาแทนได้หมด → ระบบตกปลากลายเป็นตัวกินรายได้ตัวเอง (กับดักคลาสสิกของเกม F2P)

### ✅ ตกปลาใช้สกุลเงินของตัวเองแยกออกมา

เสนอชื่อ: **"เกล็ด" / "เหรียญปลา"** (สรุปชื่อภายหลัง)

| | Jil (เติมเงิน) | เกล็ด (ตกปลา / ฟรี) |
|---|---|---|
| ได้มาจาก | เติมเงิน | ตกปลา |
| ใช้ซื้อ | คอสตูมพรีเมียมทั้งหมด | ของในสายตกปลาเท่านั้น |
| ตัวอย่างของที่ซื้อได้ | สี / หน้า / อาวุธ / หมวก | คันเบ็ด (เพิ่มพลัง) / เหยื่อ / ของแต่งตู้ปลา |

**เส้นที่ห้ามข้าม:** เกล็ดซื้อคอสตูมพรีเมียมไม่ได้ / Jil ไม่จำเป็นต่อการตกปลาเก่งขึ้น (ไม่ pay-to-win)

---

## 3. ตกปลาให้อะไรแทน Jil (3 เส้นที่ไม่ชนการขายคอสตูม)

### เส้นที่ 1 — สกุลเงินตกปลา + สายรางวัลของตัวเอง
ตกปลาได้ "เกล็ด" → ซื้อคันเบ็ด/เหยื่อ/ของแต่งตู้ วนอยู่ในสายตกปลา ไม่แตะ Jil
คนตกปลาเก่งขึ้นได้ฟรี **สำคัญ** เพราะถ้าคนจ่ายตันตั้งแต่ต้น มันจะไม่รั้งใคร

### เส้นที่ 2 — ของ exclusive ที่เงินซื้อไม่ได้
ล็อกคอสเมติกบางชิ้นให้ปลดจากตกปลาเท่านั้น
เช่น "ตกเลเจนดารี่ครบ 10 → มงกุฎทะเล"
**ไม่ชนของในร้าน Jil เพราะมันไม่ได้ขาย** เป็นของอวดล้วนๆ → คนอยู่นานขึ้น → สุดท้ายจ่ายเงิน

### เส้นที่ 3 — พลิกให้ตกปลาเป็น "ที่ใช้ Jil" ไม่ใช่ "ที่ได้ Jil" ⭐
แทนที่จะปล่อย Jil ออกมา ให้ตกปลา **สร้าง demand ใหม่ให้ Jil**:
- สกินคันเบ็ดสวยๆ (คนละอันกับคันเพิ่มพลังที่ซื้อด้วยเกล็ด — อันนี้เพิ่มความสวย) → ซื้อด้วย Jil
- ตู้ปลา / ของแต่งตู้เวอร์ชันพรีเมียม → ซื้อด้วย Jil
- พอมีปลาให้โชว์ คนอยากได้ตู้สวยมาอวด → ตกปลาไปสร้าง demand ให้ Jil

> **จุดกลับหัว:** ตกปลาที่ทำถูกวิธี ไม่ลดยอดขาย Jil แต่เพิ่มเหตุผลให้คนจ่าย Jil

**ลำดับทำ:** เริ่มเส้นที่ 2 อย่างเดียวก่อน (ปลดของจากความสำเร็จ ไม่ต้องมีสกุลเงินที่สองด้วยซ้ำ) แล้วค่อยเติมเกล็ด (เส้น 1) และของพรีเมียม (เส้น 3) ทีหลัง

## 4. สถาปัตยกรรม 2D/3D (แยกส่วนให้สะอาด)

โมเดลดึงจาก Unity (GLTF export) — ระบบตกปลา **ไม่ผูกกับ rig/animation ตัวละคร**

### การแบ่งเลเยอร์
- **เลเยอร์ 3D** ตอบคำถามเดียว: "อยู่ใกล้จุดตกปลา + หันเข้าน้ำไหม" → โชว์ prompt (กด F)
- **กดปุ่ม** → ยิง event ไปหาโมดูลตกปลา 2D (self-contained ไม่ต้องรู้ว่าข้างนอกเป็น 3D)
- **โมดูล 2D** เล่นอนิเมชัน → คืนผลลัพธ์ (ปลาอะไร/ความหายาก) → เข้า inventory

ตัวละครแค่ **นั่งท่าถือคันเบ็ดค้างไว้** ระหว่างมินิเกมรัน ไม่ต้องมีอนิเมชันพิเศษ

### การ detect จุดตกปลา — อย่าฉลาดเกิน
อย่าใช้ raycast เช็คน้ำ (เปราะ) → ใช้ **trigger zone มองไม่เห็น** วางไว้ที่จุดตกปลา นิยามเป็น data ล้วนๆ:
- พิกัด + loot table ต่อจุด (บ่อกลางคืน / แม่น้ำ / ทะเล มีตารางปลาต่างกัน)
- เพิ่มมิติได้ฟรี (กลางคืนออกปลาเรืองแสง ฯลฯ) ไม่แตะโมเดลเลย

### Feedback อยู่ที่ 2D ไม่ใช่ท่า 3D
**อย่าพึ่งท่าดีใจ 3D** (rig อาจไม่มีท่านั้น) → เอาอารมณ์ไปไว้ที่ 2D ที่คุมเอง 100%
- คอมิกเอฟเฟกต์: BAM / POW / MISS / BOOM (อ่านอารมณ์ได้ทันที ดูสนุกกว่าท่า 3D)
- กรอบ/แสงเรืองสีตามความหายาก, คอนเฟตตี, ตัวหนังสือ "LEGENDARY!" กระแทกลงมา
- เสียง fanfare คนละแบบต่อความหายาก (ได้ผลต่อความรู้สึกมากสุดต่อแรงที่ลง)
- ถ้า rig มี emote อะไรก็เล่นเป็นของแถม (ทำเป็นระบบ emote กลางไปเลยจะเอาไปใช้ที่อื่นต่อได้)

## 5. ผลลัพธ์ตัดสินที่ Server (สำคัญ)

เกมมีแชท = มีหลายคน + ไอเทมมีมูลค่า → **ห้ามให้ client สุ่มเอง**

- **Server สุ่มผล** (ติด/หลุด, ปลาอะไร, ความหายาก) แล้วส่งผลกลับ
- **Client แค่เล่นอนิเมชัน 2D ตามผลที่ได้** (โชว์ 50% → BOOM/MISS ตามที่ server ชี้ขาด)
- ไม่งั้นแก้ค่าใน memory ให้ 100% หรือเสกไอเทมได้

> หมายเหตุ: อันนี้คนละเรื่องกับการกันบอท/กัน AFK (ที่ตกลงว่าไม่ทำ) — นี่คือกันการปั๊มไอเทม ซึ่งจำเป็นถ้าไอเทมเทรด/มีมูลค่าได้

---

## 6. กลไกในมินิเกม 2D

- **ความหายาก:** ธรรมดา → หายาก → เอปิก → เลเจนดารี่
  ⚠️ **ใช้ tier + สีกรอบเดียวกับคอสตูมในร้าน** (เขียว/น้ำเงิน/ม่วง/ทอง + ป้าย EPIC)
  ปลาหายากกรอบม่วง ต้องรู้สึกเป็นความหายากเดียวกับหมวกกรอบม่วง → รู้สึกเป็นเกมเดียว
- **โอกาสจับ %** โชว์ให้เห็น (มีลุ้น มีจังหวะพลาด "ปลาหลุด!") → สร้างเรื่องคุย
- (ออปชัน) จับจังหวะกดตอนเข็มเข้าโซนแบบ Stardew / เล็งจุดโยนเบ็ด — ทำใน overlay 2D ได้หมด
- จุดตกปลาหมุนเวียนตามเวลา/สภาพแวดล้อม (กลางคืน/ฝน มีปลาต่างกัน) → เหตุผลกลับมาเช็ก

## 7. ระบบปลดล็อกด้วยการออนไลน์ (ตัวรั้งคนตัวจริง)

เปลี่ยนคันเบ็ดจาก "ของที่ซื้อ" เป็น "ของที่ต้องอยู่ถึงได้"

- **ออนไลน์สะสมครบ 2 ชม./วัน → รับคันเบ็ดฟรี** (วันละ 1 ครั้ง)
- **หรือ ชวนเพื่อน 5 คน** (ผ่านลิงก์ + ผูก Google + กันบัญชีปลอม) → ปลดทันที

### ทำไมเวิร์ค
- คนใหม่ต้องอยู่ในแชท 2 ชม. ก่อนตกปลาได้ → ระหว่างนั้นเห็น world มีชีวิต เห็นคนคุย → โอกาสติดแล้วอยู่ต่อ
- "วันละ 1 ครั้ง" = เหตุผลกลับมาทุกวัน แบบเนียน (เป็น progression ไม่ใช่ daily reward แข็งๆ)
- 2 ทางเลือก (อยู่เอง / ชวนเพื่อน) = **growth loop** ที่ป้อนตัวเอง

### ⚠️ จุดต้องระวัง
- อย่าให้ "การอยู่" น่าเบื่อจนคนเปิด tab ทิ้งแล้วไปทำอย่างอื่น — จะได้แค่ตัวเลข online ไม่ได้ชุมชนจริง
  → มีอะไรเล็กๆ ที่ทำให้ "อยู่แล้วสนุกกว่าเปิดทิ้ง" เช่น มี activity/คุย แล้วแถบเดินเร็วขึ้นนิดหน่อย / มีอะไรให้ลุ้นระหว่างนั้น ไม่ใช่แค่แถบโหลดเปล่าๆ
- **ยังไม่สรุป:** ปลดคันเบ็ดครั้งแรกแล้วตกได้ฟรีตลอด (ด่านแรกเข้าเกม) **หรือ** ต้องออนไลน์ทุกวันถึงตกได้วันนั้น (daily gate — แรงกว่าแต่กดดันกว่า)

## 8. ตลาด/ราคาปลา — เริ่มคงที่ก่อน

### เริ่มต้น: ราคาคงที่ (แนะนำ)
ปลา tier ไหนขายได้เท่าไรตายตัว — เข้าใจง่าย เทสง่าย ไม่ต้องรอ volume
(ปลาขายได้เป็น **เกล็ด** ไม่ใช่ Jil — ตามกฎข้อ 2)

### ของเติมตอนโต: ตลาดราคาผันผวน
ราคาขึ้นลงตาม supply (ขายเยอะ = ราคาร่วง แล้วค่อยฟื้น)
- **ข้อดี:** สร้างเรื่องคุยในแชท ("ปลามังกรราคาตกแล้ว อย่าเพิ่งขาย") / เพิ่มมิติให้ปลา
- **แต่:** เป็นระบบ "หนัก" อาจดึงโฟกัสจากการคุยไปเฝ้ากราฟ + ต้องมี volume คนขายมากพอราคาถึงขยับมีความหมาย
- **สรุป:** เก็บไว้เปิดตอนคนตกปลาเยอะจนราคานิ่งเริ่มน่าเบื่อ จะมาถูกจังหวะกว่า และมีข้อมูลจริงตั้ง balance

## 9. ต่อยอด (เฟสหลัง)

- **ตู้ปลา / กำแพงถ้วยรางวัล** — เอาปลามาโชว์ ไม่ขาย เป็น prestige คนละแกนกับคอสตูมบนตัว → สร้าง demand ตู้พรีเมียม (Jil)
- **Fishdex** สมุดสะสมปลา → ผูกกับปลดคอสเมติกเฉพาะ ("ตกทูน่าทอง → ปลดสกินอาวุธทอง")
- **ทางเลือก ขายหรือเก็บ** — ปลาเลเจนดารี่: ขายเอาเกล็ด หรือเก็บโชว์ในตู้? ความลังเลนี้ทำให้คนคิด+คุยกัน
- **ทัวร์นาเมนต์ตกปลาตามเวลานัด** — ดึงคนเข้าพีค / **ลีดเดอร์บอร์ด** / ประกาศกลางแชทเมื่อใครได้เลเจนดารี่ (hype)
- เชื่อมมินิเกมอื่นเป็นวงจร (ปลูกพืช → เหยื่อ → ตกปลา → ทำอาหาร → ...) ถ้าจะขยายในอนาคต

## 10. Checklist ก่อนปล่อยจริง

- [ ] **ล็อกสกุลเงิน:** ปลาขายได้เป็นเกล็ด (ไม่ใช่ Jil) — decision แรกสุด
- [ ] Server เป็นคนสุ่มผล / client เล่นอนิเมชันตามผล
- [ ] tier ความหายาก + สีกรอบตรงกับคอสตูมในร้าน
- [ ] เลือกโมเดลปลดคันเบ็ด: ครั้งเดียวจบ vs daily gate
- [ ] เริ่มด้วยราคาคงที่ (เก็บตลาดผันผวนไว้ทีหลัง)
- [ ] เริ่มด้วยเส้นรางวัลที่ 2 (ปลดของจากความสำเร็จ) ก่อนทำสกุลเงินเต็มระบบ
- [ ] มีของสายตกปลาที่ขายด้วย Jil (สกินคัน/ตู้พรีเมียม) เพื่อหนุนยอด Jil
